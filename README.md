# Customer Segmentation using K-Means Clustering

## 📌 Overview

This project is part of my Machine Learning Internship at Prodigy Infotech.

The objective of this project is to segment customers of a retail store into distinct groups based on their annual income and spending behavior using the K-Means Clustering algorithm.

Customer segmentation helps businesses better understand their customers and develop targeted marketing strategies.

---

## 🎯 Objective

To implement a K-Means Clustering model that groups customers into different segments based on:

* Annual Income (k$)
* Spending Score (1-100)

---

## 📂 Dataset

Dataset Used: Customer Segmentation Dataset

Dataset Information:

* Total Records: 200
* Total Features: 5

Features:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1-100)

Selected Features for Clustering:

* Annual Income (k$)
* Spending Score (1-100)

---

## 📊 Exploratory Data Analysis

The following steps were performed:

1. Dataset Loading and Inspection
2. Feature Selection
3. Elbow Method Analysis
4. Customer Segmentation using K-Means Clustering
5. Cluster Visualization
6. Cluster Center Analysis

---

## 🤖 Model Used

### K-Means Clustering

K-Means is an unsupervised machine learning algorithm used to group similar data points into clusters.

The Elbow Method was used to determine the optimal number of clusters.

Optimal Number of Clusters:

* K = 5

---

## 📈 Results

The K-Means model successfully segmented customers into five distinct groups based on their annual income and spending patterns.

These clusters can help businesses:

* Identify high-value customers
* Understand customer purchasing behavior
* Design targeted marketing campaigns
* Improve customer retention strategies
